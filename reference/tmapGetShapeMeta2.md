# Internal method that extracts more meta data from shape objects

Internal method that extracts meta data from shape objects

## Usage

``` r
tmapGetShapeMeta2(shp, smeta, o)
```

## Arguments

- shp:

  the shape

- smeta:

  meta (from
  [`tmapGetShapeMeta1()`](https://r-tmap.github.io/tmap/reference/tmapGetShapeMeta1.md))

- o:

  the list of options
