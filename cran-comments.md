## Release notes

This is a documentation-only patch release (4.4-1). It revises a number of
help-page titles shown in `help(package = "tmap")` to be more informative and to
remove duplicate titles; no code, user-visible behavior, or API has changed.

I am submitting sooner than the usual interval after 4.4 (released 2026-06-14)
because these documentation fixes are required for a revision of our Journal of
Statistical Software submission describing the package, and the corrected titles
need to be available on CRAN for the article to proceed. I hope this shorter
interval is acceptable given that the change is limited to documentation.
